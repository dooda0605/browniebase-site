계정 및 데이터 삭제 안내 – Badapong English

최종 수정: 2026년 5월 13일

Badapong(이하 "앱")은 사용자가 언제든지 자신의 계정과 관련 데이터를 삭제할 수 있도록 보장합니다. 본 페이지는 삭제 절차와 삭제 대상 데이터를 안내합니다.

1. 삭제 방법

방법 A — 앱 내에서 직접 삭제 (권장)

  1. Badapong 앱 실행
  2. 설정 → 계정 → 계정 삭제 탭
  3. 확인 다이얼로그에서 삭제 선택
  4. 즉시 모든 사용자 데이터가 서버에서 삭제됩니다.

방법 B — 이메일 요청

앱 내 삭제가 어려운 경우 (앱 분실, 기기 변경 등):

  • 이메일: support@browniebase.com
  • 제목: "계정 삭제 요청"
  • 본문에 포함: 사용 중인 이메일 주소(Google/Apple 로그인 시), 또는 익명 사용자 식별자(앱 설정에서 확인 가능)
  • 처리 기간: 영업일 기준 7일 이내

방법 C — 앱 삭제 (익명 사용자)

익명으로만 사용 중인 사용자는 앱을 삭제하면 다음과 같이 처리됩니다:

  • 기기 내 저장된 익명 식별자 및 사용 설정 즉시 삭제
  • 서버에 저장된 익명 데이터(즐겨찾기 등)는 90일 이내 자동 폐기

2. 삭제되는 데이터

계정 삭제 시 다음 데이터가 즉시 영구 삭제됩니다:

  • 익명 기기 식별자 (UUID)
  • 사용자 프로필 (Google/Apple 로그인 시 연결된 이메일)
  • 즐겨찾기 spot 목록
  • 작성한 후기 (텍스트, 별점, 활동 유형)
  • 후기에 첨부한 사진
  • 좋아요 기록
  • 사용 설정 (언어, 선호 활동)

3. 보관되는 데이터

아래 데이터는 법령상 보관 의무가 있어 익명화 후 일정 기간 유지됩니다:

  • 서버 접속 로그 (3개월, 보안·법적 분쟁 대응)
  • 집계 분석 데이터 (사용자 식별 정보 제거된 통계 — DAU, 화면 조회수 등)

위 데이터는 개인을 식별하지 않으며, 보관 기간 경과 후 자동 폐기됩니다.

4. 외부 서비스 데이터

Badapong은 외부 서비스(Supabase, Mapbox, Klook 등)와 일부 데이터를 공유합니다. 계정 삭제 시 Badapong 서버의 데이터는 즉시 삭제되며, 외부 서비스의 데이터 보관 정책은 각 서비스의 개인정보처리방침을 참고하시기 바랍니다.

참고: Klook 예약 기록은 Badapong이 아닌 Klook 서버에 보관됩니다. Klook 예약 데이터 삭제는 Klook 측에 별도 요청해야 합니다. (Klook 개인정보처리방침)

5. 삭제 후 영향

  • 같은 Apple/Google 계정으로 다시 로그인하면 새로운 사용자로 처리됩니다.
  • 이전 즐겨찾기·후기·사진은 복구되지 않습니다.
  • 익명 사용자의 경우 같은 기기에서 앱을 다시 설치해도 새로운 익명 식별자가 생성됩니다.
